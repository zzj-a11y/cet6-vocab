# 答题进度追踪 — 设计说明书

**日期**: 2026-05-29
**状态**: 已确认

## 概述

答题页单词和选项之间新增进度环组件，显示用户答对单词的进度。

## 进度规则

- **统计范围**：只统计答对的单词（单选题选对 / 多选题全对）
- **去重**：同一单词答对多次只计 1 次
- **公式**：`完成数 / 总单词数 × 100%`
- **持久化**：`completedIds` 数组存入 localStorage
- **重置**：管理页提供"重置进度"按钮

## UI 组件

- 位置：英文单词和选项之间
- 进度环：SVG 圆环，indigo→green 渐变，显示已完成数字
- 文字：`总词数` `已完成` `还需`
- 对齐：居中

## 文件改动

- `js/app.js`：新增 `completedIds`、`addCompleted()`、`clearCompleted()` 方法
- `js/quiz.js`：单选/多选答对时调用 `App.addCompleted()`，渲染进度环
- `js/wordManager.js`：管理页新增"重置进度"按钮
- `css/style.css`：进度环样式
