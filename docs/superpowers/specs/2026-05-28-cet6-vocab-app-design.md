# CET-6 六级单词背诵程序 — 设计说明书

**日期**: 2026-05-28
**状态**: 已确认

## 概述

一个类似百词斩的六级词汇背诵网页应用。纯前端实现，浏览器打开即用。

## 技术方案

纯原生 HTML/CSS/JS，零依赖，零构建工具，直接双击 `index.html` 运行。数据存 localStorage。

## 功能列表

### 1. 答题（看英文选中文）
- 显示英文单词，4 个中文选项（1 个正确 + 3 个随机干扰项）
- 选对：绿色高亮，1 秒后自动下一题
- 选错：红色高亮正确答案，单词自动进入错题本
- 答题时可点收藏按钮将单词加入收藏本

### 2. 错题本
- 展示所有答错的单词
- 可进入错题专项练习
- 专项练习完成后错题本自动清空

### 3. 收藏本
- 展示用户收藏的单词
- 可浏览复习
- 支持取消收藏

### 4. 单词管理
- 表格展示所有单词
- 在线增删改
- 支持上传 .docx/.txt 文件批量导入

### 5. 底部 Tab 栏
- 答题 / 错题本 / 收藏本 / 管理 四个入口切换

### 6. 设备布局切换
- 顶部三个按钮：iPhone 15 Pro Max (430px) / iPad Air 11" (820px) / MacBook Neo 13" (1280px)
- 即时切换，内容自适应

## 数据流

- 词汇表：加载时合并默认词汇（`data/words.js`）+ localStorage 中用户自定义词汇
- 错题本：答错时写入 localStorage，练习完成后清空
- 收藏本：用户收藏时写入 localStorage，取消时移除
- 单词管理：增删改操作同步更新 localStorage

## 答题逻辑

1. 从词汇范围（全部/分组）中随机抽取单词
2. 为该单词生成 3 个随机干扰选项（从其他单词的释义中抽取）
3. 4 个选项随机排列显示
4. 用户选择后判断对错，记录结果
