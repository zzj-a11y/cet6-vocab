# 三设备专属响应式布局 — 设计说明书

**日期**: 2026-05-29
**状态**: 已确认

## 概述

替换当前简单的宽度拉伸式"适配"，为三种设备设计专门的 UI 布局。

## iPhone 15 Pro Max (430px)

- 单栏全宽布局
- 单词顶部居中，选项纵向堆叠
- 按钮触控区 ≥56px
- 收藏 + 上一题按钮靠下方排列
- 顶部简洁 header + FAB 圆钮
- 底部 Tab 栏常驻

## iPad Air 11" (820px)

- 左右分栏布局
- 左侧 (45%)：单词 + 进度环 + 统计小卡片（错题数、收藏数）
- 右侧 (55%)：2×2 网格选项按钮
- 顶部信息栏（标题 + 进度 + FAB）
- 底部操作条（收藏 + 上一题 + Tab 文字导航）

## MacBook Neo 13" (1280px)

- 三栏布局
- 左侧 (170px)：垂直导航栏（4 个 Tab + 进度环 + FAB 切换）
- 中间 (flex)：答题主区（单词 + 2×2 选项 + 键盘快捷键提示 + 操作按钮）
- 右侧 (200px)：统计面板（大圆环 + 正确率/今日目标/连续打卡进度条 + 最近复习标签云）

## 技术方案

- CSS：每种设备有专属样式块，通过 `[data-layout]` 选择器激活
- JS：无需改动 app.js 核心逻辑，仅需在 `bindLayoutButtons` 中同步更新
- 数据：统计面板复用已有的 `completedIds`、`wrongIds`、`favoriteIds`
